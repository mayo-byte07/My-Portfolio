1. **Clone the repository**
   ```sh
   git clone https://github.com/mayo-byte07/My-Portfolio.git
   cd My-Portfolio
   ```

2. **Install dependencies**
   ```sh
   npm install
   ```

3. **Start the development server**
   ```sh
   npm run dev
   ```
   The application will be available at `http://localhost:5173`
